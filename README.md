An enterprise-level static webpage engineered to not only teach Web Accessibility (A11y) rules but to strictly demonstrate them in its own architecture. Built with a premium "Midnight Tech" dark theme, focusing on zero cognitive load and flawless performance.

## 🚀 Technical Achievements
- **Perfect Lighthouse Scores:** Achieved 100/100 in Accessibility, Best Practices, and SEO. 
- **W3C Validated:** Zero markup errors.
- **Strict Semantic HTML:** Zero generic `<div>` soup. Built entirely with proper landmark tags (`<header>`, `<main>`, `<section>`, `<footer>`) for native screen-reader support.
- **Modern CSS Architecture:** - Zero inline styles or IDs used for styling.
  - Strict adherence to the DRY principle using CSS variables (`:root`).
  - Fluid macro-layouts using **CSS Grid** and micro-alignments using **Flexbox**.
- **WCAG Compliant:** Strictly maintains a 4.5:1 color contrast ratio across the dark theme and uses relative units for scalable typography.
- **Performance Optimized:** Includes explicit dimensions and lazy-loading for `.webp` assets to prevent Cumulative Layout Shift (CLS).

## 🛠️ Tech Stack
- HTML5 (Semantic)
- CSS3 (Custom Properties, Grid, Flexbox)
